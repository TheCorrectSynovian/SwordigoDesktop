# Swordigo Formats

## gamedata.gdata

Message 1 = Item

Fields:

1 = Item Type
2 = Internal ID
3 = Display Name
4 = Stat Text
5 = Description

Known Types:

1 = Consumable
2 = Weapon
3 = Armor
4 = Trinket
5 = Quest Item

---

## player.gstate

3 = Current Scene
4 = Spawn Point
9 = Map

11 = Inventory Item
15 = Unlocked Ability
16 = Equipped Ability
17 = Equipped Trinket
