Release Notes

Track the evolution of the Swordigo Runtime Environment.

June 2026

v2.0r — "Vanguard"

Current Stable

Complete overhaul of the graphics pipeline and migration to the new JNI bridge architecture.

  • Added: Full SSAO and God Rays implementation.
  • Added: Support for high-refresh-rate displays (144Hz+).
  • Improved: ELF relocation performance (reduced boot time by 40%).
  • Fixed: Texture corruption on AMD/Mesa drivers.
May 2026

v1.8.2 — Bug Fixes

Maintenance release focusing on stability and input compatibility.

  • Fixed: Controller axis drift on specific gamepads.
  • Improved: OpenAL buffer management.
  • Added: Experimental Wayland support.
April 2026

v1.0.0 — Initial Release

The first public release of the Swordigo Runtime Environment.

  • Initial ARM emulation core based on Unicorn.
  • GLES 1.x translation layer.
  • Basic SDL2 platform support.
