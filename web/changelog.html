Release Notes

The full development history of Swordigo Desktop — from first boot to current stable.

v6.0

June 22, 2026 Current Stable

The Wastelands release. Two game-breaking freezes killed, SRE expanded to 30+ hooks, full GUI stack, and a complete PostFX pipeline.

🎉 FIXED: Wastelands spinlock — the infamous game-breaking freeze that locked the main thread in an infinite ARM loop
🎉 FIXED: Death freeze — instant checkpoint respawn via SRE hook, bypassing the ad/prompt system entirely
🔧 SRE expanded to 30+ active hooks (was 17 in v5.0)
🖼️ Full GUI DrawRect stack — 8 native hooks for buttons, labels, frames, and sliders
💚 GameSceneView::Update recovered — HP bar, mana orb, coin counter, and damage flash all functional
🪙 Smart coin bar with shop-aware auto-hide logic
💾 Save Editor built into the launcher — edit gold, HP, items, and quest flags
📦 Asset Viewer for browsing PVR textures, PNG sprites, and audio files
✨ PostFX pipeline: 6 presets — Cinematic, Retro, Fantasy, Noir, Ethereal, Atmospheric
🛡️ Crash safety: luaD_throw + ProgramPanic + __cxa_throw interception
⚠ Known issue: Bolt/timer misbehavior — some enemies fire projectiles too fast due to ARM timer drift.

v5.0

June 21, 2026

The SRE release. Introduced libsre.so as a guest-side hook library, replacing binary patching with proper function interception.

🚀 libsre.so introduced — 17 active hooks into libswordigo.so at runtime
🎵 Full music system replacement via OpenAL command interface
💀 Death/respawn system — skip ads, instant checkpoint teleport
📊 HUD reimplementation through GameSceneView::Update
⚛️ Non-atomic string ops — STXR elimination for x86 compatibility
📡 Player stats (HP, mana, gold, position) exported to host for HUD rendering

v4.5r

June 20, 2026

Launcher tools and rendering improvements. The Save Editor was revamped and integrated directly into the launcher UI.

💾 Save Editor revamped and moved into the launcher
🔩 ARM64 entity loop fix — corrected iteration over game objects
🎮 GPU draw call batcher for reduced overhead
✨ PostFX pipeline groundwork — SSAO and god rays shaders

v4.0r

June 19, 2026

Architecture overhaul. PolyMC-style launcher with instance management, ARM64 emulation via Unicorn Engine, and full SDL3 migration.

🖥️ PolyMC-style launcher with instance manager and version selector
🦾 ARM64 emulation via Unicorn Engine (replaced ARM32 core)
🪟 SDL3 migration complete — modern windowing, input, and audio

v1.0

June 14, 2026

The beginning. First successful boot of Swordigo on a Linux desktop — proof of concept with ARM32 emulation and basic GLES translation.

🏁 Initial release — first native Linux port of Swordigo
⚙️ ARM32 emulation core (Unicorn Engine)
🔺 Basic GLES 1.x translation layer
🎮 SDL2 platform support
