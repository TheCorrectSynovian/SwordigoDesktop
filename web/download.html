Installation Instructions

1. Install the Package

Use your distribution's package manager to install the downloaded file.

# For RPM (Fedora/RHEL)
sudo dnf install ./swordigo-desktop-*.rpm

# For DEB (Debian/Ubuntu)
sudo apt install ./swordigo-desktop_*.deb

2. Run the Bootstrap Script

The bootstrap script will download the necessary game assets and set up the environment.

chmod +x instx.sh
./instx.sh --bootstrap

3. Launch the Game

Run the launcher script to start the game with the SRE runtime.

./run_swordigo.sh
Dependency Note

Ensure you have libsdl2 (or libsdl3), libopenal, and mesa-libGL installed if they are not automatically pulled by the package manager.
