Swordigo Desktop

A professional Linux-native port of the classic action-adventure. Powered by the Swordigo Runtime Environment (SRE) — a high-performance ARM compatibility layer.

Get Started Read the Docs
⚙️

SRE Core

Custom ARMv7 emulator bridge with zero-overhead syscall translation and native GLES mapping.

🎨

Enhanced Visuals

Integrated PostFX pipeline featuring SSAO, God Rays, and high-fidelity texture filtering.

🐧

Linux Native

First-class support for Wayland/X11, SDL3 input, and native PulseAudio/PipeWire integration.

Quick Start

Clone the repository and run the bootstrap script to get started with the development environment.

git clone https://github.com/TheCorrectSynovian/SwordigoDesktop.git
cd SwordigoDesktop
./instx.sh --bootstrap
./run_swordigo.sh
